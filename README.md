# AT32F011 Firmware Library Package

## Overview

**AT32F011 Firmware Library** is a firmware package which contains the device drivers for all of the peripherals for AT32F011.
   * This library provides a set of examples for each peripheral and some application such as the IAP and sLib utilities.
   * AT32F011_periph_driver : device drivers 
   * AT32F011_periph_examples : examples for AT32F011
   * AT32F011_periph_template : AT32F011 project template for Keil and IAR IDE tools 
   * AT32F011_board : AT-START-F011 board setting functions 
   * AT32F011_middlewares_i2c_application_library : middlewares for I2C application 
   * UTILITIES_examples : some applications such as FreeRTOS, sLib and IAP.
